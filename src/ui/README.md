# Howto

- Run ```install_deps.sh```
- Change port (default: 8088) in ```package.json``` if necessary
- Change dispatcher adress in ```src/ui/src/app/app.component.css``` if necessary: by default it's ```localhost:8088```
- Run ```start.sh```
